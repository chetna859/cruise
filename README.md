# Cruise Ship Management System

A comprehensive web-based management system for cruise ship operations, featuring multiple user roles and service bookings.

## Features

### User Services
- **Resort Booking** - Book luxury resort accommodations
- **Movie Booking** - Reserve movie theater seats
- **Fitness Center** - Access gym facilities and book sessions
- **Beauty Salon** - Schedule beauty and wellness appointments
- **Catering Services** - Order food and dining services
- **Party Hall** - Reserve event spaces for celebrations
- **Stationery Shop** - Purchase office and personal supplies

### User Management
- User registration and authentication
- Shopping cart functionality
- Order management and tracking
- Booking confirmations

### Admin Roles
- **Admin Dashboard** - Overall system management
- **Manager Dashboard** - Service oversight and reporting
- **Supervisor Dashboard** - Operational monitoring
- **Head Cook** - Kitchen and catering management

## Getting Started

1. Open `index.html` in a web browser
2. Register a new account or login with existing credentials
3. Browse available services and make bookings
4. Admin users can access their respective dashboards
